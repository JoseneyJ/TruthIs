# Truth Is
A Christian confessional blog website that provides a supportive space for users to explore content related to personal life struggles. The platform aims to inspire, encourage, and reassure readers that they are not alone in their spiritual journey with Jesus.

## Tasks
    - Responsive design for various devices.
    - Category-based article organization.
    - Search functionality for articles.
    - Commenting system (if applicable).
    - RSS feed for new posts.
    - AI chat box feature 

## Tools used: 
    - Frontend: HTML, CSS (Tailwind CSS), JavaScript (React)
    - Backend: Node.js (Express)
    - Database: MongoDB
    - Hosting: Vercel
