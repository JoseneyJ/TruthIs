# Truth Is
A Christian confessional blog website designed to provide a supportive space for users to explore content related to personal life struggles. The platform aims to inspire, encourage, and reassure readers that they are not alone in their spiritual journey with Jesus.


## Tasks
    - Responsive design for various devices.
    - Category-based article organization.
    - Search functionality for articles.
    - RSS feed for new posts.
    - AI chat box feature 

## Tech Stack:
    - Frontend: HTML, CSS (Tailwind CSS), JavaScript (React)
    - Backend: Node.js (Express)
    - Database: MongoDB
    - Hosting: Vercel

## Current Improvements
* Fixing category upload issues (currently limited to “Lifestyle”)
* Improving UI/UX for better engagement and accessibility
